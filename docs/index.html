An AI asset-QA gate: it retrieves the brand-book rules relevant to each generated asset and returns an evidence-based APPROVE / REJECT, then ranks the approved ones pairwise. Same idea as agentic-qa-gate, pointed at generated assets. This is the live output on 5 synthetic assets (2 clean, 3 with planted violations).
